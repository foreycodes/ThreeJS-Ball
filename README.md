<!-- GETTING STARTED -->

### Prerequisites

Ensure you have downloaded and installed Three.JS and GSAP.
* npm
  ```sh
  npm install three gsap
  ```

### Installation


1. Clone the repo
   ```sh
   git clone https://github.com/foreycodes/ThreeJS-Sphere.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the program
   ```sh
   npm start
   ```
